## Showcase Your Project

Are you using Stashtab? We'd love to feature you!

**To be featured:**

1. Open a [GitHub Issue](https://github.com/TRC-Crypto/stashtab/issues/new?template=feature_request.md)
2. Include:
   - Project name and description
   - Website URL
   - Screenshots (if possible)
   - How you're using Stashtab
   - Any customizations you've made
