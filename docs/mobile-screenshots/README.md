# Mobile App Screenshots

This directory contains screenshots of the Stashtab mobile app for documentation and showcase purposes.

## Screenshot Guidelines

When adding screenshots:

1. **Format**: PNG or JPG
2. **Resolution**: Minimum 1080x1920 (portrait) or 1920x1080 (landscape)
3. **Naming**: Use descriptive names like `home-screen.png`, `deposit-flow.png`
4. **Organization**: Group by feature or screen

## Current Screenshots

Screenshots will be added as the mobile app is developed and polished.

## Adding Screenshots

1. Take screenshots using your device or emulator
2. Save to this directory with a descriptive name
3. Update [SCREENSHOTS.md](../SCREENSHOTS.md) to reference the new screenshots

## Tools

- **iOS**: Use Xcode Simulator screenshot feature
- **Android**: Use Android Studio emulator or device screenshot
- **Expo**: Use `expo-screen-capture` or device screenshot
