# Stashtab Integration Templates

Choose the template that matches your tech stack.

## Rails Template

**Best for**: Ruby on Rails applications

```bash
git clone https://github.com/TRC-Crypto/stashtab/templates/rails-stashtab.git
```

Features:

- Rails controller integration
- View helpers
- Asset pipeline support
- Active Record models (optional)

## Django Template

**Best for**: Python/Django applications

```bash
git clone https://github.com/TRC-Crypto/stashtab/templates/django-stashtab.git
```

Features:

- Django views integration
- Template tags
- Django REST framework support
- Database models (optional)

## Express Template

**Best for**: Node.js/Express applications

```bash
git clone https://github.com/TRC-Crypto/stashtab/templates/express-stashtab.git
```

Features:

- Express routes
- TypeScript support
- Middleware integration
- Database integration (optional)

## Which Template Should I Use?

| Framework | Use Case                              |
| --------- | ------------------------------------- |
| Rails     | Existing Rails app, Ruby team         |
| Django    | Existing Django app, Python team      |
| Express   | New Node.js app, TypeScript preferred |
